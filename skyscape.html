Skyscape

C++ DirectX 11 Game

Introduction

Skyscape is a tech demo in the form of a game, written within a C++ framework using DirectX 11 and nVidia PhysX.

What I learned

The framework contains uv mapping, collions, raycasting, triggers, skyboxes, image and text rendering, skinned animations, cloth, particle effects, light baking and post-processing shaders.
I implemented all these components prior to applying them to build the game.
Theoretically, I learned all about the graphics rendering pipeline.
