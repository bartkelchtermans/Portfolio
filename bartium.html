Bartium

Web handler

Introduction

In the last 30 years, the internet evolved fundamentally, however the way we access it has stayed the same. With cloud computing, and so many web-based applications, we often don't need a browser because our online behavior isn't browsing.
Bartium is a way to access the web in a new way, a desktop client of anything.
I built it using the amazing Electron technology by Github.

What I learned

The entire project is built as a web-based application, so I learned a lot about Javascript and web development. I thoroughly studied how web browsers work and how any web-based interaction happens.
Furthermore, I got to know all about Electron and I started using Github for this project.

What now?

Bartium is still under active development to stay up to date with the Chromium framework, improve performance and fix bugs. Nobody can tell where its future lies.
