F-14

Mechanical Rigging Demo

Introduction

For this project I wanted to learn all about rigging, how it works and how it's done in the industry.
I started with an existing mesh of an F-14 and divided up all the vital parts. I proceeded to study reference in order to learn how it works in real life and how to best implement it in my project.

What I learned

The pipeline of rigging a model, from analyzing reference, to setting up the rig, to adding handlers and eventually animating.
How different chains (IK/FK) work, animating in 3ds Max and importing a skinned mesh in Unreal engine.
